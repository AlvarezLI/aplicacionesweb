# NUVEA — Tienda Premium Online

**Autor:** Alvarez Luis Ignacio Jesús

## Descripción

NUVEA es una tienda online de estilo premium que ofrece productos seleccionados en tres categorías: Moda, Electrónica y Hogar. El proyecto es el trabajo práctico de la primera entrega de la materia de Desarrollo Web, con foco en la estructura HTML y diseño CSS estático de múltiples páginas interconectadas.


## Estructura del proyecto

aplicaciones-web/
├── index.html          
├── login.html          
├── register.html        
├── css/
│   └── styles.css       
├── pages/
│   ├── moda.html       
│   ├── electronica.html 
│   └── hogar.html       
└── README.md


## Páginas

| Página | Descripción |
| `index.html` | Home con navbar, hero y grilla de las 3 categorías |
| `login.html` | Formulario de inicio de sesión (email + contraseña) |
| `register.html` | Formulario de registro (nombre, apellido, email, contraseña, fecha de nacimiento) |
| `pages/moda.html` | Categoría Moda con grilla de productos |
| `pages/electronica.html` | Categoría Electrónica con grilla de productos |
| `pages/hogar.html` | Categoría Hogar con grilla de productos |

## Tecnologías

- HTML5 semántico
- CSS3 con variables custom (sin frameworks externos)
- Google Fonts: Cormorant Garamond + DM Sans
